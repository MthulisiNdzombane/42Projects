La classe Vertex crée un point dans l'espace possédant 3 coordonnées 'x', 'y' et 'z'. Ainsi que 2 paramètres optionnels: une couleur (class Color) et une caractéristique 'w'.

Exemple:
$color = new Color(array("red" => 100, "green" => 30, "blue" => 215));
$point = new Vertex(array("x" => 10, "y" => 10, "z" => 10, "color" => $color));
